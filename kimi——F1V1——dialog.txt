  • 🇸🇦 = S + A 的区域指示符

  这样可以在所有系统上显示为国旗（在支持的系统上）或字母代码（在不支持的系统上）。

  📱 三、原理说明

  局域网访问原理

  ┌─────────────────┐         WiFi/网线          ┌─────────────────┐
  │   你的电脑       │  ←════════════════════→   │   手机/平板      │
  │  192.168.1.5    │      同一局域网内           │  192.168.1.8    │
  │                 │                           │                 │
  │  ┌───────────┐  │                           │  浏览器输入：    │
  │  │ npm run   │  │                           │  192.168.1.5    │
  │  │ dev --host│  │                           │  :5173          │
  │  │ 端口 5173  │  │                           │                 │
  │  └───────────┘  │                           └─────────────────┘
  └─────────────────┘

  关键点：

  1. 所有设备必须在 同一个 WiFi/路由器 下
  2. 需要知道电脑的 内网 IP（不是公网 IP）
  3. Windows 防火墙需要允许端口访问
